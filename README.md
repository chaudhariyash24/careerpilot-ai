# CareerPilot AI 🚀

An AI-powered career guidance and productivity platform designed to help students and professionals make smarter career decisions through personalized recommendations, skill analysis, learning roadmaps, and AI-driven assistance.

## 🌟 Features

### AI Career Guidance

* Personalized Career Recommendations
* AI Career Assistant
* Goal-Based Career Planning

### Resume & Skill Analysis

* ATS Resume Analyzer
* Skill Gap Analysis
* Resume Improvement Suggestions

### Learning & Growth

* Personalized Learning Roadmaps
* Course Recommendations
* Progress Tracking Dashboard

### Career Development

* Job Recommendations
* Aptitude Tests
* Performance Analytics
* Achievement & Leaderboard System

### User Management

* Secure Authentication
* User Profiles
* Certificate Management
* Activity Tracking

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/chaudhariyash24/careerpilot-ai.git
```

Navigate to project directory:

```bash
cd careerpilot-ai
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
PORT=3000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
```

Run the application:

```bash
npm start
```
